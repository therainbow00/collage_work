1. List any headers in the get-project-gutenberg.txt request here.

answer: Host: www.gutenberg.org

2. Reconstruct the URL from the host and the path provided in
   get-project-gutenberg.txt.

answer: http://www.gutenberg.org/ebooks/

3. What is the HTTP request verb being used in the
   get-project-gutenberg.txt request?

answer: GET

4. What is the HTTP response code?

answer: 302

5. What is the URL specified in the returned Location header?

answer: www.gutenberg.org
